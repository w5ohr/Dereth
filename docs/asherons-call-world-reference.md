### Recurring inter-town "Foundry" / "Residential Hall" portal lines
- **Effigy Foundry → Cragstone (26.0N 48.4E):** from Zaikhal, Al-Jalima, Uziz, Khayyaban, Hebian-To, Lin, Kryst (the "everyone routes to Cragstone" line).
- **Mammet Foundry → Baishi (49.3S 62.9E):** from Plateau Village, Stonehold, Sawato, Tufa, MacNiall's Freehold.
- **Mannikin Foundry → Nanto (~51–52S 82E):** from Al-Arqas, Samsur, Shoushi, Yaraq, Holtburg, Lytelthorpe, Rithwic.
- **Colossus Foundry → Fort Tethana (1.7N 71.2W):** from Kara, Ayan Baqur, Wai Jhou.
- **Victory Residential Halls:** Samsur/Lin ↔ Khayyaban (47.6S 26.7E). **Oriel Residential Halls:** Mayoi/Plateau ↔ Khayyaban. **Haven Residential Halls:** Uziz/Lytelthorpe ↔ Tou-Tou.
- **Capital triangle:** Holtburg ↔ Shoushi ↔ Yaraq direct portals.
- **Settlement portals:** clusters beside each town (e.g. Crater Lake 24+, Plateau 22, Stonehold 18) lead to player-housing settlements.

## 4. Vendors / merchants — types & what they sell
Core rule: **a vendor only buys items of the type it sells**. Vendors trade in **pyreals**, wear an **apron**, show as **yellow radar dots**.

| Type | Sells / trades |
|---|---|
| Weaponsmith | weapons (melee & missile) |
| Armorer / Blacksmith | armour, shields, smithing goods |
| Provisioner / Grocer / Shopkeeper / Merchant / Peddler | food, drink, packs, supplies, ammo |
| Farmer / Spice Merchant | foodstuffs, spices, reagents |
| Healer | healing kits/services |
| Jeweler | gems, jewelry, trinkets, tools |
| Scrivener / Scribe | spell scrolls, components, paper (~155% sell) |
| Archmage / Mastermage / Geomancer | components, mana stones, magical goods |
| Barkeeper | drinks & rumors (many buy nothing) |
| Contract Broker / Town Crier | contracts, rumors, directions |

**Economics:** sell 100–190% (Scribe ~155%, many shops 135%); buy 50–95% (commonly 90%). Some vendors use alternate currencies (MMDs, trade notes, Luminance).
